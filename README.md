# Resources about the usage of Blender in Science, Academia, and Industry
This is a curated list of the literature, scientific papers, blog posts and articles about free and open-source 3D and 2D multimedia editor software [Blender](https://blender.org) being used in various fields. Feel free to submit your suggestions via pull requests. This list was initially created as a supplementary material to the paper [Survey of FOSS 3D/2D Graphics Software Blender Usage in Science, Academia, and Industry](...). The paper was created as a source that can be cited to support the usage of Blender as a full-fledged scientific, academic, or industrial tool. Citation of the paper is available [here](CITATION.cff). Feel free to cite it if you find the resources helpful.
